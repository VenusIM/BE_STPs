### Study Summary
#### > TDD (테스트 주도 개발)
- 프로덕션 코드보다 테스트 코드를 먼저 작성하는 개발 방법
- 기능 동작 검증

#### > BDD (행위 주도 개발)
- 행위에 대한 테스트 코드를 작성하는 개발 방법
- 유저 시나리오(Given, When, Then 구조) 동작을 검증
